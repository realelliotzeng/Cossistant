When writing `discord-announcement`:

- Write plain text only. No JSON, no embeds, no headings.
- Make it read naturally in Discord and allow line breaks.
- Use a short opener, then 2-4 concise release details, then a CTA.
- Include the changelog URL `{{CHANGELOG_URL}}`.
- Keep the total message within Discord limits and suitable for one channel message.
- Prefer the strongest widget and developer-experience improvements first.
